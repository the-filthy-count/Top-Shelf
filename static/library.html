Top-Shelf —

A self-hosted media filing tool and Stash companion. Perceptual-hash matching against StashDB, ThePornDB, and FansDB automatically identifies adult video files and organizes them into a Kodi-compatible library with NFO sidecars, posters, and fanart.

What it does

  • Watches source folders, probes videos with FFmpeg, matches phashes through StashDB → ThePornDB → FansDB.
  • Routes files into Studio Series or Performer paths based on matched metadata.
  • Writes Kodi-style .nfo sidecars with cast, genres, studio, and date.
  • Downloads poster / fanart art and triggers Jellyfin, Plex, Emby, or Stash library scans.
  • Integrates with Prowlarr + NZB / torrent clients for end-to-end Discover → Download → File flows.
  • Tracks library health: duplicates, orphaned metadata, missing folders, filing history.
